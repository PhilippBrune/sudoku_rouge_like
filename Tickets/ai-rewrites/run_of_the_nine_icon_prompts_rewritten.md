# Run of the Nine -- Icon Production Prompt Pack

This document contains production-ready prompts to generate all icons for the game **Run of the Nine** using a consistent Kyoto Zen Garden pixel-art style.

Implementation status (current repository):

- Placeholder/generated icons are integrated at `Assets/Resources/GeneratedIcons`.
- Runtime loading paths use `Resources.Load("GeneratedIcons/<icon_name>")`.
- Use this prompt pack when replacing placeholders with final hand-authored art.

## Assumptions/Clarifications

1. Icon prompts are for pixel art icons, 64x64 pixels.
2. Icons should have a clean sprite atlas style, Kyoto zen garden aesthetic.
3. Icons should have soft calming colors, Japanese temple design language, moss stone textures, warm lantern lighting, and subtle shading.
4. Icons should have a crisp readable silhouette, minimal but expressive, roguelike game UI icon, centered object, transparent background, no text, no frame, pixel perfect, consistent lighting from top-left, soft shadows, limited palette, and elegant and readable.

## Global Style Prompt (Use for ALL Icons)

All icons should adhere to the following style: pixel art icon, 64x64 pixels, clean sprite atlas style, Kyoto zen garden aesthetic, soft calming colors, Japanese temple design language, moss stone textures, warm lantern lighting, subtle shading, crisp readable silhouette, minimal but expressive, roguelike game UI icon, centered object, transparent background, no text, no frame, pixel perfect, consistent lighting from top-left, soft shadows, limited palette, and elegant and readable.

## Main Menu Icon Prompts

Icons for the main menu should adhere to the following style: pixel art icon, 64x64, soft lighting, Kyoto garden atmosphere, and centered object.

## Roguelike Item Icon Prompts

Icons for roguelike items should adhere to the following style: pixel art icon, 64x64, soft lighting, centered object, and subtle shading.

## Puzzle UI Icon Prompts

Icons for puzzle UI elements should adhere to the following style: pixel art icon, 64x64, soft lighting, centered object, and subtle shading.

## Rarity Frame Icon Prompts

Icons for rarity frames should adhere to the following style: pixel art icon frame, soft lighting, centered object, and subtle shading.

## Sudoku Grid Icon Prompts

Icons for Sudoku grids should adhere to the following style: pixel art icon, 64x64, soft lighting, centered object, and subtle shading.

## Sprite Atlas Layout

The recommended atlas size is 512x512 for the entire atlas and icon size is 64x64. Each row represents a different category of icons.

## Extra Prompt Modifier

Icons should be styled with a strong silhouette, minimal noise, high contrast, centered object, and pixel perfect.
