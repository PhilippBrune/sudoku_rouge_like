# JIRA-0006 — Puzzle Region Layout Variants & Box-Size Modifier

**Created:** 2026-03-07

## Summary

Add region layout variety to sudoku puzzles:
1. **Box-orientation modifier** for 6×6 and 8×8 boards — swap between horizontal and vertical rectangular box layouts (e.g. 2×3 ↔ 3×2).
2. **Alt** for 5×5 and 7×7 boards — at least two distinct templates per size, randomly selected at level start so consecutive puzzles of the same size use different layouts.
3. **Fix 7×7 contiguity bug** — the current 7×7 template (from JIRA-0005) has 5 of 7 regions with disconnected cells in the last row.

## Related Tickets

- **JIRA-0005**: Established region templates for 5–9 sizes; 7×7 template had correct cell counts (7 each) but broken contiguity.

## Requirements

### R1 — Region variant support in generation pipeline
- `LevelConfig` gets `RegionVariant` field.
- `PuzzleGenerationRequest` gets `RegionVariant` field.
- `SudokuGenerator.BuildRegionMap()` accepts variant parameter.
- `SudokuGenerationService.BuildRegionMap()` passes variant through.
- `RunDirector.BuildLevelConfig()` randomly assigns a variant per level.
- Fallback path in `RunDirector.StartLevel()` also uses the variant.

### R2 — Box-orientation modifier for 6×6 and 8×8
- 6×6 variant 0: 2×3 boxes (2 rows, 3 cols) — existing behavior.
- 6×6 variant 1: 3×2 boxes (3 rows, 2 cols).
- 8×8 variant 0: 2×4 boxes — existing behavior.
- 8×8 variant 1: 4×2 boxes.
- 9×9: always 3×3 (no variant).

### R3 — Alternative templates for 5×5
- Template 0 (existing): hourglass/bowtie shape.
- Template 1 (new): diagonal-band shape.
- Selected by `variant % 2` at level start.

### R4 — Alternative templates for 7×7 + contiguity fix
- Template 0 (new, replacing broken one): horizontal wave bands — all 7 regions contiguous.
- Template 1 (new): vertical wave bands (90° rotation of template 0) — all 7 regions contiguous.
- Both validated for cell counts (7 per region) and contiguity (BFS).
- Selected by `variant % 2` at level start.

### R5 — Irregular jigsaw template for 6×6
- Variant 2: 6 non-rectangular, orthogonally contiguous hexomino regions (6 cells each).
- Staircase L-shapes tiling the 6×6 grid.
- Validated with BFS contiguity check.

### R6 — Irregular jigsaw template for 8×8
- Variant 2: 8 non-rectangular, orthogonally contiguous octomino regions (8 cells each).
- Symmetric staircase pattern with mirrored L-shapes in top/bottom halves.
- Validated with BFS contiguity check.

### R7 — Irregular jigsaw template for 9×9
- Variant 2: 9 non-rectangular, orthogonally contiguous nonomino regions (9 cells each).
- Diagonal wave staircase spanning full grid; includes U-shaped region in bottom rows.
- Validated with BFS contiguity check.
- `RunDirector.BuildLevelConfig()` updated: `_random.Next(2)` → `_random.Next(3)` to include jigsaw variant.
