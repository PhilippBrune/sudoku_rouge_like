# Run of the Nine --- Icon Generation Guide (Fooocus)

This document defines image prompts and file naming conventions for
generating all icons used in the game using Fooocus.

The goal is to maintain visual consistency, pixel‑art clarity, and easy
asset management.

------------------------------------------------------------------------

# Global Style Settings (Use For Every Icon)

Style - Pixel Art - 64x64 resolution - Clean silhouette - Minimal
shading - Soft Kyoto garden color palette - Calm greens, stone grays,
lantern gold accents

Theme Inspiration - Kyoto zen gardens - Ryoan‑ji rock garden
minimalism - Kinkaku‑ji temple elegance

Lighting - Soft top light - Slight glow for magical items

Background - Transparent - No shadows outside icon bounds

------------------------------------------------------------------------

# Naming Convention

icon\_`<category>`{=html}*`<name>`{=html}*`<rarity>`{=html}.png

Examples

icon_relic_combo_common.png\
icon_relic_lantern_legendary.png\
icon_class_gardener.png\
icon_ui_settings.png
