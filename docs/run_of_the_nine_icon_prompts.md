# Run of the Nine -- Icon Production Prompt Pack

This document contains production-ready prompts to generate all icons
for the game **Run of the Nine** using a consistent Kyoto Zen Garden
pixel-art style.

Implementation status (current repository):

- Placeholder/generated icons are integrated at `Assets/Resources/GeneratedIcons`.
- Runtime loading paths use `Resources.Load("GeneratedIcons/<icon_name>")`.
- Use this prompt pack when replacing placeholders with final hand-authored art.

------------------------------------------------------------------------

# Sprite Atlas Layout

Recommended atlas size:

-   Atlas: 512x512
-   Icon size: 64x64
-   Grid: 8x8

Example structure:

Row1: Start \| Resume \| Tutorial \| Meta \| Modes \| Items \| Options
\| Credits\
Row2: Quit \| Hint \| Undo \| Redo \| Error \| Correct \| Charm \| Time\
Row3: Reveal \| Shield \| Vision \| Rune \| 5x5 \| 6x6 \| 7x7 \| 8x8\
